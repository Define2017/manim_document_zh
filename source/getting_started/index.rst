Todd Zimmerman整理了 `一个非常好的教程`_ ，介绍了 ``manim`` 的入门，该教程已更新为在Python3.7上运行。注意你可能需要
把 ``from big_ol_pile_of_manim_imports import *`` ，改成 ``from manimlib.imports import *`` 。
